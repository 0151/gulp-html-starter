# Gulp HTML/SCSS/ES6 starter

## Requirements
[Node.js](https://nodejs.org/)

## Installation
```
npm i gulp-cli -g
npm i
```

## Usage
- ```$ gulp``` - start dev server
- ```$ gulp build``` or ```gulp build --production``` - build project
